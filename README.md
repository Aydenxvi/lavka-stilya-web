# 🛍 Лавка Стиля — Premium Aesthetic Web Drop '26

Короче, это фронтенд для стильного и вайбового интернет-магазина одежды. Сначала юзер попадает на футуристичную форму регистрации с эффектом матового стекла (Glassmorphism), а после клика бесшовно залетает на главную витрину с кучей шмоток и плавными 3D-анимациями.

Сделано чисто на базе **HTML5, CSS3 и ванильного JavaScript** — никаких тяжелых фреймворков типа React, всё летает на ультрах прямо из коробки.

## ✨ Что тут по фичам?

- **Трендовый дизайн**: Подкрутил глубокие неоново-пастельные градиенты, кастомные шрифты (`Inter`) и прозрачные стеклянные плашки. Смотрится дорого и эстетично.
- **Интерактивные 3D-карточки**: Карточки товаров реагируют на курсор! Благодаря CSS-перспективе они плавно наклоняются в пространстве, когда ты на них наводишь. 
- **Абсолютная автономность**: Все фотки товаров переведены на супер-стабильные глобальные сервера (Wikimedia), так что визуал прогрузится на 100% у любого челика даже без включенного VPN.
- **Живой фронтенд на JS**: 
  - Форма регистрации не просто перезагружает страницу, а плавно тушит экран входа и разворачивает магаз.
  - Кнопка «В корзину» рабочая — сверху на хэдере тикает счетчик товаров.
  - При добавлении шмотки снизу вылетает красивое push-уведомление (тост).

## 🛠 Технологический стек

- **HTML5** — семантическая разметка (хэдер, мейн, карточки).
- **CSS3** — кастомные CSS-переменные (`:root`), Flexbox, CSS Grid для сетки товаров и `perspective` для 3D-движений.
- **Vanilla JS** — чистый код для обработки кликов, анимации переходов и логики корзины.

## 🚀 Как запустить этот движ у себя?

Всё максимально просто, справится даже первокурсник:

1. Скопируй весь код сайта в файл и назови его, например, `index.html`.
2. Если хочешь, чтобы на форме регистрации была твоя личная аватарка, закинь фотку в ту же папку и переименуй её в `orkhan.jpeg`. (Если фотки нет — не парься, сработает авто-заглушка и подтянется стильная иконка).
3. Просто дважды кликни по файлу `index.html` — он откроется в любом браузере (Chrome, Safari, Edge). Всё, можно тестить и кайфовать!
